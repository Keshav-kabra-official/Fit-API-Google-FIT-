
	****  Android Fit Application Using Google Fit API  ****

==================================================================================================

• Counts the number of steps that the user has taken so far using Sensor Client.
• Number of steps are displayed on the screen in a TextView as well as using a Toast message.

• Requires the Google Permission to access the Gmail Account of the user.

• Count Steps even when NOT connected to the internet.

• Find the Images Folder in the repository to look at the overview of the app.
